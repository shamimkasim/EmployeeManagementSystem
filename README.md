# Employee Management System

## Project Overview
The Employee Management System is designed to manage employee records for a fictitious company. This system allows for managing employee details such as name, surname, email, document number, phone numbers, and manager relationships. It is built with .NET 8 for the backend and plans for a React/Angular frontend.

## Features
- **CRUD Operations**: Create, Read, Update, and Delete employees.
- **Authentication**: Secured with JWT authentication.
- **Document Validation**: Unique and obligatory document number validation.
- **Multiple Phone Numbers**: Support for handling multiple phone numbers per employee.

## Technologies Used
- **.NET 8**: Backend API
- **React/Angular**: (Planned) Frontend
- **Docker**: Containerization
- **JWT**: Authentication
- **SQL Database**: Managed in Docker
